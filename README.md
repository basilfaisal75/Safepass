SafePass is a desktop-based password manager that I created for my final-year Computer Science (Forensics) thesis. Its goal is to assist users construct passwords that are both secure and memorable. The fundamental idea behind SafePass is to address the widespread issue of weak or reused passwords by integrating personal security questions with strong cryptographic algorithms. Instead of depending just on random password generation, the application generates memorable passwords based on user responses to security questions, ensuring they satisfy modern security standards. SafePass is written in Python and has a basic Tkinter-based graphical interface to make it accessible to non-technical users. 
From a security standpoint, the application places a strong emphasis on securing saved credentials: bcrypt is used to securely hash and validate passwords, while AES encryption is utilised to protect sensitive data at rest within a local SQLite database. Password strength is measured using entropy analysis to determine how resistant generated passwords are to brute-force attacks. Throughout development, SafePass was evaluated to ensure that its fundamental features were reliable, usable, and correct, and basic attack scenarios were simulated in a controlled environment to examine its resilience to typical threats such as credential theft and keylogging. Legal and ethical aspects were also addressed, ensuring that user data is managed securely and properly. Overall, SafePass demonstrates a practical balance between usability and security, while showcasing applied skills in cybersecurity, cryptography, and secure software development.
